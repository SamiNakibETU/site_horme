## 🏠 1. Modifier la Page d'Accueil

### Où?
Cliquez sur **🏠 Accueil** dans le menu.

### Qu'y a-t-il à l'accueil?

1. **1️⃣ Vidéo + Titre** - Vidéo de fond et "Cie. Horme"
2. **2️⃣ Photos qui défilent** - Le carrousel horizontal
3. **3️⃣ Bandeau bleu** - "Le corps comme langage premier"
4. **4️⃣ Texte "La compagnie"** - Bloc texte principal
5. **5️⃣ Photos** - Section "La compagnie en images"
6. **6️⃣ Créations** - Titres avant la liste des spectacles

### Comment modifier?

**📸 Changer les photos du carrousel (section 2️⃣)?**

1. Cliquez sur **"2️⃣ Photos qui défilent"** (onglet en gris)
2. Vous voyez une liste de photos
3. Pour **ajouter** une photo:
   - Cliquez le bouton bleu **"Ajouter"**
   - Choisissez votre photo (depuis votre ordi ou la médiathèque)
   - Cliquez OK

4. Pour **changer l'ordre**:
   - Attrapez l'icône **⋮⋮** à gauche d'une photo
   - Glissez-la vers le haut/bas

5. Pour **supprimer** une photo:
   - Cliquez le **X** à droite de la photo

**⚠️ Important**: 
- ❌ Ne mettez PAS les photos de personnes seules ici! 
- ✅ C'est pour les photos de groupe/danse seulement!
- Après chaque changement, cliquez le bouton **"Publier"** en haut à droite!

## 👥 2. Modifier la Page Présentation

### Où?
Cliquez sur **👥 Présentation**.

### Qu'y a-t-il?

- **Surtitre** (ex: "La compagnie")
- **Titre principal** (ex: "Cie. Horme")
- **Texte d'introduction** (texte riche: possibilité de gras, italique, listes)
- **Portraits** - Joséphine, Louise, et autres danseuses

### Comment ajouter des danseuses?

1. Allez dans **👥 Présentation**
2. Trouvez la section **"👥 Portraits"**
3. Cliquez **"Ajouter"** (bouton bleu)
4. Remplissez:
   - **📸 Photo portrait** - Photo carrée du visage
   - **✍️ Nom** - "Joséphine" ou "Louise"
   - **🏷️ Rôle** - "Co-fondatrice" ou autre
   - **📝 Biographie 1** - Texte sur sa carrière, style
   - **📝 Biographie 2** - Suite (optionnel)
   - **🏷️ Titre formation** - "Formation" ou "Cursus"
   - **📝 Texte formation** - Où elle a étudié, écoles, influences

5. Cliquez **"Publier"** en haut à droite

## 🎭 3. Créer/Modifier une Création (Spectacle)

### Où?
Cliquez sur **🎭 CRÉATIONS** dans le menu.

### Comment créer une nouvelle création?

1. Cliquez le bouton **"Créer"** (en haut)
2. Remplissez:
   - **✍️ Titre** - "Tout ce qui tremble", "Rann", etc.
   - **URL (auto)** - Se crée tout seul depuis le titre
   - **🏷️ Sous-titre** - Court descriptif (optionnel)
   - **📁 Type** - Festival / Performance / Événement / Résidence
   - **📅 Année** - Ex: 2025
   - **📍 Lieu** - Ex: "Paris, Théâtre Douze"
   - **📷 Photographes** - Noms séparés par des virgules
   - **📝 Description** - Texte long présentant le spectacle
   - **🖼️ Image de couverture** - Photo forte pour les listes
   - **📸 Galerie photos** - Les photos de la page spectacle
   - **🎬 Vidéo** - Optionnel (fichier MP4)

3. **Pour ajouter photos à la galerie**:
   - Trouvez la section **"📸 Galerie photos"**
   - Cliquez **"Ajouter"**
   - Choisissez une photo
   - Vous pouvez remplir:
     - **📷 Crédit** - Nom du photographe
     - **📝 Description** - Ce qu'on voit

4. **Pour changer l'ordre des photos**:
   - Attrapez **⋮⋮** à gauche
   - Glissez-les dans l'ordre souhaité

5. Cliquez **"Publier"** !

## 🔗 4. Modifier le Menu de Navigation

### Où?
Cliquez sur **⚙️ CONFIGURATION DU SITE** → **🔗 Menu de navigation**

### À quoi sert ce menu?
C'est le menu en haut à droite du site avec le bouton "Création" et ses sous-menus.

### Comment modifier?

**Structure prévue**:

```
Création (bouton principal)
├─ Tout ce qui tremble (groupe avec 3 sous-projets)
│  ├─ Jardiniers Montrouge
│  ├─ Regard du Cygne
│  └─ Théâtre Douze
├─ Rann
├─ Khorrmos
└─ Bastille Design Center
```

Dans Sanity, vous verrez:
- **"Création — groupe 1 (surtitre)"** = "Tout ce qui tremble"
- **"Création — liens groupe 1"** = Les 3 sous-projets (Jardiniers, Regard, Théâtre Douze)
- **"Création — liens groupe 2 (sans surtitre)"** = Rann, Khorrmos, Bastille

Pour ajouter un lien:
1. Cliquez **"Ajouter"** (bouton bleu)
2. Remplissez:
   - **Libellé** - "Rann", "Khorrmos", etc.
   - **URL (chemin)** - "/creation/rann", "/creation/khorrmos", etc.
3. Cliquez **"Publier"**

## 📸 5. Gestion des Photos (Médiathèque)

### Où accéder aux photos?

En haut du Sanity Studio, vous voyez plusieurs onglets. Cherchez **"Médiathèque"** ou **"Media".

### Comment?

1. Cliquez sur l'onglet **"Médiathèque"**
2. Vous voyez toutes les photos du site
3. Vous pouvez:
   - **Ajouter** une nouvelle photo (bouton "Upload")
   - **Chercher** une photo (barre de recherche)
   - **Organiser** les photos en dossiers

### Conseils pour les photos:

✅ **BON**:
- Format: JPG ou PNG
- Taille minimale: 1920 pixels de large (accueil), 1200 pixels (galeries)
- Couleurs: bien contrastées, intéressantes
- Cadrage: pas coupées aux bords, marges respectées
- Qualité: nette, bien éclairée

❌ **À ÉVITER**:
- Photos coupées au bord
- Photos trop compressées (floues)
- Trop petites (moins de 800px)

## 💾 À RETENIR: PUBLIER VOS CHANGEMENTS

**⭐ TRÈS IMPORTANT**: Après chaque modification, vous DEVEZ cliquer sur **"Publier"** (bouton en haut à droite).

Sans cliquer "Publier", vos changements restent en brouillon et ne s'affichent PAS sur le site!

### Étapes:
1. Modifier quelque chose (texte, photo, etc.)
2. Cliquez le bouton bleu **"Publier"**
3. Attendez que le message "Publié" s'affiche ✅
4. Allez voir sur le site pour vérifier!

## ❓ Questions Fréquemment Posées

### Q: Comment voir le site pendant que je modifie?

**R**: Dans Sanity Studio, cliquez sur l'onglet **"Présentation"** pour un aperçu en direct.

### Q: Je me suis trompée en modifiant. Puis-je revenir en arrière?

**R**: 
- Si vous n'avez PAS publié: cliquez **"Abandonner"** (bouton gris à côté de Publier)
- Si vous AVEZ publié: cliquez l'icône **"Historique"** (horloge) en haut pour voir les anciennes versions

### Q: Les photos ne s'affichent pas correctement

**R**: Vérifiez que:
- L'image n'est pas coupée à la création
- Vous avez cliqué **"Publier"**
- Vous avez attendu 5-10 secondes puis rafraîchi le site (Ctrl+F5)

### Q: Je veux ajouter une nouvelle page (pas juste modifier l'existant)

**R**: Contactez le développeur - les nouvelles pages demandent de modifier le code.

### Q: Comment changer le footer (pied de page)?

**R**: Allez dans **⚙️ CONFIGURATION DU SITE** → **👣 Pied de page**. Remplissez les textes, coordonnées, réseaux sociaux.
